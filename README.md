# Full Stack Blog App

A full-stack blogging platform built while learning the MERN stack. Users can create, edit, and delete blog posts.

## Features

* Create/Edit/Delete Blogs
* Responsive UI
* MongoDB Database Integration

## Tech Stack

* Node.js
* Express.js
* MongoDB


## Future Improvements

* Comments System
* Likes & Bookmarks
* Rich Text Editor
* Image Upload Support
